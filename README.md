# 🚀 Anime Offis Ads SDK

El SDK oficial de **Anime Offis AF GDL México** para la integración de publicidad dinámica basada en el estándar VAST 3.0.

## 📌 Descripción
Este sistema permite a desarrolladores y colaboradores de la comunidad de anime integrar anuncios en video de forma aleatoria y automatizada en sus reproductores web.

## 🔗 API Endpoint
Endpoint global para obtener anuncios en formato XML:
`https://ads.animeoffis.com`

## 🛠️ Instalación y Uso
1. Copia el archivo `ads-helper.js` y `styles.css` en tu proyecto.
2. Importa la lógica en tu reproductor.
3. Configura el evento de finalización para proceder con tu contenido principal.
